# Simple Quiz App

A basic Python console application that presents a series of multiple-choice questions, tracks the user's score, and displays results at the end.

## Features

- Multiple-choice questions with predefined answers
- Score tracking throughout the quiz
- Final results display with percentage score
- Simple console-based interface

## Requirements

- Python 3.x

## Usage

Run the script to start the quiz:

```
python simple_quiz.py
```

The app will present questions one by one. Enter the number corresponding to your answer choice. At the end, you'll see your total score and percentage.
