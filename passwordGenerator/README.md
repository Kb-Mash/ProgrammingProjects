# Password Generator

A simple Python script to generate secure random passwords based on user preferences.

## Features

- Customizable password length
- Option to include numbers
- Option to include special symbols
- Secure randomization using Python's `random` module

## Requirements

- Python 3.x

## Usage

Run the script and follow the prompts to specify:
- Password length
- Whether to include numbers
- Whether to include symbols

Example:
```
python password_generator.py
```

- Randomization techniques
- String manipulation
- User input validation
