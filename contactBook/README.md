# Contact Book

A simple command-line contact book that lets you store, search, delete, and persist contacts in a local file.

## Description

This project is a small Python application that demonstrates basic file I/O and data management concepts. It allows you to keep a list of contacts with names and phone numbers, search for contacts by name, delete contacts, and save/load your contact list from disk.

## Features

- Add a contact (name + phone number)
- Search contacts by name
- Delete contacts
- Persist contact list to a file (save/load)
- Simple command-line interface

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kb-Mash/ProgrammingProjects.git
   cd ProgrammingProjects/contactBook
   ```

2. Ensure Python 3 is installed on your system.

## Usage

Run the contact book program:

```bash
python contact_book.py
```

Follow the on-screen prompts to manage your contacts.

```

4) Save contacts

Enter name: Alice
```
