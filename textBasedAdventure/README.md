# Text-Based Adventure Game

A simple interactive text-based adventure game built in Python where players make choices that determine the story's outcome.

## Description

This game immerses players in a choose-your-own-adventure style narrative where every decision branches the story in different directions. Experience multiple endings based on your choices in this classic text-based gaming format.

## Features

- Interactive storytelling with branching narratives
- Multiple choice-based decision making
- Multiple possible endings
- Simple command-line interface
- Replayable gameplay

## Requirements

- Python 3.x

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/Kb-Mash/ProgrammingProjects.git
   cd ProgrammingProjects/textBasedAdventure
   ```

2. Ensure Python 3.x is installed on your system.

## How to Run

Run the game using Python:

```bash
python text_based_game.py
```

## How to Play

1. Start the game by running the script
2. Read the story text presented to you
3. Make choices by typing the corresponding number or letter when prompted
4. Your decisions will affect the story's outcome
5. Try different choices to discover all possible endings!

## Game Structure

The game follows a decision tree structure where:
- Each scene presents a situation and choices
- Player input determines the next scene
- Different paths lead to different conclusions
- Some choices may lead to game over scenarios
